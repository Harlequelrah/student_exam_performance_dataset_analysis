Ce dataset utilisé dans ce repository pour pratiquer l'analyse de donnée.

# 3. Utilisation

- **cloner le repository** :

```cmd
git clone https://github.com/Harlequelrah/student_exam_performance_dataset_analysis
```

- **créer un environnement viruel** :

```cmd
python -m venv .venv
```

ou

```cmd
uv venv
```


- **installer les dépendances** :

```
pip install -r requirements.txt
```

```cmd
uv sync
```
